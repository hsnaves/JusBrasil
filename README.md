JusBrasil
=========


Sample code used to give the talk at JusBrasil.


Build
-----

To build the code, you will simply type:

    $ make

Then run:

    $ ./plsa --help

or

    $ ./hmm --help

To compile the python module, type:

    $ cd python
    $ python setup.py build_ext --inplace

and to run:

    $ python test_plsa.py -h
